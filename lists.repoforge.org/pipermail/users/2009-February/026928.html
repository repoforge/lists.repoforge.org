I'm not familiar with yum, but isn't this protectbase or priorities 
blocking you?

In any case on x86_64 systems I recommend adding the following line to 
~/.rpmmacros (create this file if needed):
%_query_all_fmt %%{name}-%%{version}-%%{release}.%%{arch}

it'll add the arch to all rpm queries, often very informative.
