I use Nagios for system monitoring, and needed to monitor the local
Subverson server. I only found one already existing graceful tool for this,
and it wasn't RPM wrapped. So I've bundled it at
https://github.com/nkadel/nagios-plugins-check_svn_repo-srpm/

Note that I renamed the source tool 'check_svn.txt' as 'check_svn_repo' to
distinguish it from the Subverson checking tools at NagiosExchange which
check local working copies.
