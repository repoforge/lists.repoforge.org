Now it would help if you could verify if both apt, old-yum and new-yum 
have the same problems. Because both apt and yum cache the 
md5sum/sha1sums, but they use a different cache. So if only one of both is 
corrupt, it's an issue with the tool. Otherwise it either is a 
disk-problem or disk-cache problem, or something else.
