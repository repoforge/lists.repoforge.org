 * python >= 2.5
 * twisted >= 8.1
 * twisted-web >= 8.1
 * pyopenssl
 * simplejson (if python < 2.6)
 * setuptools
 * gettext
 * pyxdg
 * geoip-database (optional)
 * libtorrent >= 0.14.5

=== UIs ===
 * chardet

=== Gtk ===
 * python-notify (libnotify python wrapper)
 * pygame
 * pygtk >= 2.12
 * librsvg
 * xdg-utils

=== Web ===
 * mako

Everything looks more or less reasonable, but I guess we're stuck with
