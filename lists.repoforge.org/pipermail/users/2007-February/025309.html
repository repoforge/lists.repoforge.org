Installed the legacy nvidia drivers
nvidia-x11-drv-96xx-1.0.9631-1.el4.rf to support an old card:

01:00.0 VGA compatible controller: nVidia Corporation NV11 [GeForce2 MX/MX 400] (rev b2)

Got errors that X could not load modules bitmap and pcidata.  Finally
traced it to a ModulePath problem after considerable Googling.

The original file had the lines:
        ModulePath   "/usr/lib/xorg/modules/extensions/nvidia"
        ModulePath   "/usr/lib/xorg/modules"

To get it to work it was necessary to add the line
        ModulePath   "/usr/X11R6/lib/modules"
before the above.  Adding it at the end failed to help.

Hope this spares someone else with old nVidia hardware the wasted time,
and that it is possible to patch the package.

Phil
