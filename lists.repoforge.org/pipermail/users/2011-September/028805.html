Hello, I put together a standard spec file for perl module
Authen::OATH.  I use it here for verifying Google Authenticator
logins, but it works for any HOTP (RFC 4226) or TOTP (RFC 6238) based
password scheme.  I have created a pull request for it.  Please accept
and build this if there is any interest in providing this to accompany
the existing perl authentication packages:

perl-Authen-Challenge-Basic
perl-Authen-DigestMD5
perl-Authen-GoogleAccount
perl-Authen-Krb5-Admin
perl-Authen-Krb5-Simple
perl-Authen-PAM
perl-Authen-Radius
perl-Authen-SASL
perl-Authen-Smb

If there is no interest in this model, please do let me know and I'll
kill the pull request.
