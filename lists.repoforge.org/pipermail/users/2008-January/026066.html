/var/cache/davfs2
    System wide directory for cached files. Used when the file system is
    mounted by root. It must belong do group davfs2 and read, write and
    execute bits for group must be set. There is a subdirectory for
    every mounted file system. The names of this subdirectories are
    created from url, mount point and user name.
