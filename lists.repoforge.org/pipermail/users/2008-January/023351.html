Well _my opinion_ ( so don't give to much about that ;) )  is that you
should use %_bindir, whatever that is set to.  And btw there is also %
_mandir and %datadir so your files section could be:

%files
%defattr(-,root,root)
%doc README LICENSE FAQ
%_bindir/nex
%_bindir/nvi
%_bindir/nview
%_mandir/man1/nex.1.gz
%_mandir/man1/nvi.1.gz
%_mandir/man1/nview.1.gz
%_datadir/vi

Look no hardcoded paths isn't that nice :) Of course you'd have to
change the configure line too:

./configure --prefix=%_prefix --mandir=%_mandir --program-prefix=n
--disable-curses

or you could just use the %configure macro
