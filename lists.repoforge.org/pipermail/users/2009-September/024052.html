 From the description:
The Atari 2600 Video Computer System (VCS), introduced in 1977, was the most
popular home video game system of the early 1980's.  This emulator will run
most Atari ROM images, so that you can play your favorite old Atari 2600 
games
on your PC.

Source file is at:
http://prdownloads.sourceforge.net/stella/stella-3.0-src.tar.gz?download

Modified Spec file is attached.  The original spec file that comes with 
the tar file is designed for Mandrake systems.

For reference and testing purposes, various other supplemental files are 
available at:
http://www.atariage.com/2600/emulation/RomPacks/Atari2600_A-E.zip
http://www.atariage.com/2600/emulation/RomPacks/Atari2600_F-J.zip
http://www.atariage.com/2600/emulation/RomPacks/Atari2600_K-P.zip
http://www.atariage.com/2600/emulation/RomPacks/Atari2600_Q-S.zip
http://www.atariage.com/2600/emulation/RomPacks/Atari2600_T-Z.zip
