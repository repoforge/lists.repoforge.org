Hi All,

I finally got around to updating amavisd-new from rpmforge on the mail 
server today, and noticed a few missing perl deps in my setup.

I didn't capture the message, but starting the amavisd service gave a 
warning about DKIM (I think there's a new DKIM setting in 2.6.4?) and 
perl modules missing.

Anyway, installing the following fixed the warnings for me, so not sure 
if these should be package dependencies (or not?).

perl-Mail-DKIM-0.36-1.el5.rf which pulled in as deps
perl-Digest-SHA-5.47-1.el5.rf
perl-Crypt-OpenSSL-RSA-0.25-1.el5.rf

All seems to be working now (as in no more warnings), although I've not 
tested the DKIM stuff.
