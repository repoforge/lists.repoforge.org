Actually the '/usr/bin/scilab' program isn't a console program: it starts the 
following:
/usr/lib/scilab-3.1.1/bin/zterm -e /usr/lib/scilab-3.1.1/bin/scilex
You can start the console program with:
 /usr/lib/scilab-3.1.1/bin/scilex
