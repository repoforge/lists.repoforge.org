Hi all,
  I have created a new rpm package named PSCM. It
integrats
postfix, spamassassin, clamav and
mailscanner/amavisd-new. This
should help to eliminate the hassle of installing and
making these applications work together.

PSCM comes in 2 flavors:

   1. Postfix, SpamAssassin, ClamAV and MailScanner
   2. Postfix, SpamAssassin, ClamAV and Amavisd-new 

The current version of the individual application in
the rpm package is:

    * Clamav: 0.88
    * MailScanner: 4.49.7-1
    * SpamAssassin: 3.1.0
    * Postfix: 2.2.8
    * Amavisd-new-2.3.3

http://metawire.org/~pscm/index.html
