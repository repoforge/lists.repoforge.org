Possible temporary workarounds:


make sure firefox is installed
start gxine with the following command:
LD_LIBRARY_PATH=/usr/lib/firefox-1.0.7/ gxine

or 

make sure firefox is installed
add /usr/lib/firefox-1.0.7 to /etc/ld.so.conf
and run /sbin/ldconfig as root
