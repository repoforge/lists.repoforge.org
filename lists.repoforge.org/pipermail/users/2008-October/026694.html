amavisd-new installs just fine for me. I guess you are not using
protectbase or priorities yum plugin.
I have priorities plugin configured with priority=1 for base and
updates, priority=10 for rpmforge. That way the perl-File-Temp from
rpmforge is simply ignored.
http://wiki.centos.org/AdditionalResources/Repositories/RPMForge
recommends yum priorities plugin for all releases.
