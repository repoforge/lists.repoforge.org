Hi all,

I'm using the perl-XML-SAX rpm package of rpmforge on a CentOS 5.4.

Each time a perl script calls the XML-SAX module it prints:
could not find ParserDetails.ini in /usr/lib/perl5/vendor_perl/5.8.8/XML/SAX

This can be solved by creating the PerserDetails.ini file with following 
command:
perl -MXML::SAX -e 
"XML::SAX->add_parser(q(XML::SAX::PurePerl))->save_parsers()"

This is described in the faq of perl-xml:
http://perl-xml.sourceforge.net/faq/#parserdetails.ini

Can this be added to the rpm?

Thx!
