%description
C implementation of a ALAC decoder, written from reverse engineering the
ALAC
file format. Apple Lossless Audio Codec (ALAC) is a proprietary lossless
audio
compression scheme.

The alac program will not be able to handle all ALAC files. It can only
handle
mono or stereo files. It can read an ALAC stream from either a file or from
stdin, and write it as raw PCM data or as a WAV file to either stdout or a
file.
