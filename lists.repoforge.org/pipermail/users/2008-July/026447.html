I have been testing alpine with the maildir patch and have now converted 
to using maildir myself for some of my mailboxes. I also have discussed 
this off-list with the maildir author.

The new alpine build that I will release the next days will have the 
maildir patch applied. If you really want to be without the functionality 
(which in itself should not affect you if you're not using maildir anyway) 
than you can still disable it by adding to your pine.conf or .pinerc:

 	disable-these-drivers=md

Let this list know if you do find any issues with the new build, just 
in case.
