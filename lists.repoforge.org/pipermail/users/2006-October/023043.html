For those of you that don't like spam... ;-)

Info here:
http://wiki.apache.org/spamassassin/FuzzyOcrPlugin

Upstream developers are: 
Christian Holler (latest version 2.3b, but he will continue development soon
apparently)
http://users.own-hero.net/~decoder/fuzzyocr/
Jorge Valdes (developed 2.3c-j)
http://www.joval.info/proj/FuzzyOcr.html

Our CVS of spec files (for SME Server 7.0, which is CentOS4), sources and
patches are here, and are for the 2.3j version:
http://smeserver.cvs.sourceforge.net/smeserver/FuzzyOcr/
http://smeserver.cvs.sourceforge.net/smeserver/gocr/

Note that our spec files are for CentOS 4.  Some changes to the spec files
are necessary for Fedora which uses giflib, not libungif.  Details on the
dependencies are here:
http://www.joval.info/proj/FuzzyOcr-2.3j/INSTALL

Hope you find this useful!  I think we'll be keeping this up to date on our
end, but thought it might be something rpmforge would be interested in.
