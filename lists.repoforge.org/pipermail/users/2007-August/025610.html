I think the asterisk package that was in there got removed because it
was woefully out of date, and building a package for it apparently isn't
exactly straightforward (and when they did come they tended to be fairly
delayed after security updates came out, which didn't help much).

Digium actively recommends people to install from source anyway, to get
good performance out of it, so that probably discourages people from
trying to package it.
