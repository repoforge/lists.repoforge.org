Hello everyone,

we would like to contribute our Open-sharedroot [1] RPMs to the RPMforge 
repository so that more people get to know our project and will work with it.

Open-sharedroot is an open-source technology that can be used to create 
Diskless Shared Root Clusters [2] in combination with the Red Hat Cluster 
Suite [3].

We are Red Hat Advanced Partners for Clustering and the Global File System and 
we implement Open-sharedroot on daily basis in enterprise environments. So 
the project is actively maintained and will get accustomed to new RHEL/CentOS 
releases.

Currently we have two seperate yum channels [4]. One for tech-preview and one 
for stable releases.

We are planning on regular releases. Each stable release will pass an 
exhaustive QA process.

Please let me know if the RPMs suite your submission guidelines and if you 
think this contribution could be of value for RPMforge.

[1] http://www.open-sharedroot.org
[2] http://en.wikipedia.org/wiki/Diskless_Shared_Root_Cluster
[3] http://www.redhat.com/software/rha/cluster/
[4] 
http://www.open-sharedroot.org/faq/can-i-use-yum-or-up2date-to-install-the-software

Regards 

Reiner
