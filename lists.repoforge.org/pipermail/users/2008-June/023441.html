On Wed, 4 Jun 2008, Alfred von Campe wrote:

> Would it be possible to build RapidSVN for EL5?  An older version (0.9.4 vs. 
> the current 0.9.6) is in the repo for EL4, but there is nothing for EL5.

Problem on RHEL5 and newer is that the package searches for apr-config, 
while the newer apr-devel package includes a pkgconfig file instead of a 
seperate -config binary.

If you need this and want to help out, you could help by either working 
with upstream to use pkgconfig from configure, or help modify the SPEC 
file so it can also work with pkgconfig.

I have no time to investigate and I don't know if anyone else will.

I did start a new build for 0.9.6 on RHEL4 and older...
