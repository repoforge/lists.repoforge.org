Hi,

Please consider applying the attached patch to nagios.spec against
the current revision 6654.  It adds %config(noreplace) protection for
Nagios HTML directory tree %{_datadir}/nagios/.  This is to prevent RPM
upgrades from overwriting local modifications to HTML stylesheets etc.
(Currently I need to modify stylesheets/status.css every time Nagios
gets an RPM upgrade to stop Status Summaries from blinking).
