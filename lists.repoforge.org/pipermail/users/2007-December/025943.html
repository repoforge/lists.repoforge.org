Hello,

I recently noticed that ruby-shadow is only available for el5. I looked
into it and patched the SPEC to build on EL4 (and probably others), too.

The patch is not very complex and it works for me on EL4 (and still
builds on EL5).
Beware that the rpmmacros %fedora", %rhel and %centos_ver are
non-standard, so the %has_ruby_abi macro might need some adjustment to
work with DAR. However, as far as I can tell from the buildlogs it
should work out of the box.

If anyone could come up with a better solution, this would be really
great. It might be possible to define a macro that actually asks rpm
whether there is ruby(abi) on that system, but I haven't tried to figure
out how that can be done.

Andreas
