Try using the "modifyrepo" program.It can inject a file into the repo config file. example:/usr/bin/modifyrepo /var/nfsroot/dist/updateinfo/centos5-x86_64/updateinfo.xml /var/www/mrepo/centos5-x86_64/RPMS.updates/repodata
 That works for me :-) BR,Bjarne
