There are many useful packages that we don't have. :) And it's better to 
have these packaged by someone who is using them. So feel free to 
contribute SPEC files. We have set up a mailinglist for this:

	http://lists.rpmforge.net/mailman/listinfo/suggest

Where you can send SPEC files, package requests and patches. Depending on 
the contributions and the overhead this means to existing packagers, you 
will get subversion access :)

We plan to do it like this to eventually end up with subversion access for 
those people that contribute the most. It also helps us to have a 
consistent way of creating SPEC files (your first contributions will be 
actively reviewed by an existing packager when it is added to 
subversion until you get subversion access).
