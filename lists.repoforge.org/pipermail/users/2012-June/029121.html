libserf 1.1.0 is out. I'm using it because it's required for the planned
Subversion 1.8.0 release, which is entirely discarding the neon library.

There is an existing libserf 0.3.0 in Repoforge, and the update is easy to
build and test with, but the major releases are  incompatible, So I'm
including .spec files for botih libserf-1.1.0 and a libserf0-0.3.0 to
retain compatibility for anyone already using libserf 0.3.0.
