I had to make a few changes to get the latest to work.
1. Add :/usr/bin/:/usr/share/rrdtool/fonts to php setting open_basedir
2. Change cacti settings: "RRDTool Default Font Path"; "Title Font 
FileTitle Font File"; "Legend Font File"; "Axis Font File"; and "Unit 
Font File" to /usr/share/rrdtool/fonts/DejaVuSansMono-Roman.ttf

Well, a least that worked for me, YMMV.
