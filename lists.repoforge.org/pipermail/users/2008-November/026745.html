Dear fellow rpmforge users

i have updated the nagios spec in rpmforge to 3.0.5. The new version
fixes a security issue (Cross Site Request Forgery). Unluckily it also
breaks something (some command and comment submissions from the cgis
won't work). The nagios folks are already testing a patched version so i
expect a 3.0.6 release within a few days.

So once the new nagios package hits the repository you have to decide
for yourself if you want to live with the security issues (version
3.0.4) or with some command functions not working (version 3.0.5)
