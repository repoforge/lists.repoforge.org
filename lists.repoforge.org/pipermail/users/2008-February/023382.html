Hello... I'm new to CentOS and rpm's in general, but I enlisted the help
of a friend to review and fix up my spec file. After doing that, he also
submitted it to the Fedora project.

I don't know what (if any) changes need to be made in order to use this
in the rpmforge repository for CentOS 5 (or any other versions), but I
submit the spec file and src rpm file here for possible inclusion:

----------

Description: General-purpose stream-handling tool

cstream filters data streams, much like the UNIX tool dd(1).

It has a more traditional commandline syntax, support for precise
bandwidth limiting and reporting and support for FIFOs.

Data limits and throughput rate calculation will work for files > 4 GB.

http://ndim.fedorapeople.org/cstream/2.7.4-3/cstream.spec
http://ndim.fedorapeople.org/cstream/2.7.4-3/cstream-2.7.4-3.src.rpm

----------

I could potentially maintain this in the future, but it's so simple
I doubt much will be required. Also, I'm not (yet) sure what's
involved in that.

Thanks for your consideration.
