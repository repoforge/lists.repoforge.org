The default config file for rkhunter-1.2.9-1.el4.rf generates a fatal
error on CentOS 4:

[root at radar1 etc]# rkhunter -c
Fatal error: can't find INSTALLDIR option in configuration file (/etc/rkhunter.conf)

Inserting the following in /etc/rkhunter.conf fixes the problem:

INSTALLDIR=/usr
