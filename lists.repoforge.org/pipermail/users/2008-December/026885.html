I have 64bit flash-plugin test packages in the EL5 x86_64 test repository. 
Let me know if you have any issues with these.

Since I now have a new (64bit) laptop, I personally will be looking more 
closely to 64bit packages on the desktop.

For those interested in test packages, the test repository usually holds 
experimental updates and updates of software that are not deemed stable by 
upstream.

One example is the newer Wine releases (1.1.x) are in the test 
repositories, while the stable Wine releases (1.0.x) are in the normal 
repositories.

Or I have recent hplip test packages for more current HP printer/scanner 
support. Things like these that would otherwise replace (important) 
upstream packages also go into the test repository.
