I've built a dkms package around the sundance network driver version LK1.20 
(http://www.icplus.com.tw/driver-pp-IP100A.html), and would like to submit 
it to RPMForge.

I currently have the ipw3945 and nvidia dkms packages installed for 
reference.  I wasn't able to name my package sundance-LK1.20-1.nodist.rf 
because rpmbuild didn't like a hyphen in my version number, but I'm sure the 
packagers here know how to get around that.

The rpm is available at 
http://greenwoodlakepilots.org/sundance-LK1.20-1dkms.noarch.rpm.  All I had 
to do to dkms-ify the driver was to add the dkms.conf file, then run the 
dkms tool to add, build, mktarball, and mkrpm.
