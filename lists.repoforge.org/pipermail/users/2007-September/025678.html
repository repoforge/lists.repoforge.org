The '-ta' only works on tarballs, not on spec files.

> then, I tried below command.
> [root at worldnet SPECS]# rpmbuild -ba havp.spec
> error: File /usr/src/redhat/SOURCES/havp- init.patch: No such file or
> directory

The havp-init.spec file can be found in the same directory in the subversion, 
see http://svn.rpmforge.net/svn/trunk/rpms/havp/
By the way you might consider building as non root.. there are some tutorials 
on that, for example at http://www.owlriver.com/tips/non-root/  according to 
google.
