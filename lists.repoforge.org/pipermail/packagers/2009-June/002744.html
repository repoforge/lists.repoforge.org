In fact the subversion-1.6.spec had this:

 	##Tag: test

which is not the same as:

 	# Tag: test

The first is a disabled comment header, the second a working one :)

Also for packages that exist in the upstream repository (like subversion) 
we prefer to set the release the 0.1, 0.2, ...

We'll do that starting from 1.6.3 again :)
