Continuing a thread on freshrpms-list, I have been maintaining a syslog-ng RPM 
for awhile now, and am interested in packaging a couple other pieces of 
software that I'm sure people would find useful, such as:

syslog-ng        http://www.balabit.com/products/syslog_ng/
wpa_supplicant   http://hostap.epitest.fi/wpa_supplicant/
jabberd (2.0)    http://jabberd.jabberstudio.org/
pmacct           http://www.ba.cnr.it/~paolo/pmacct/

These packages are currently available at:
http://www.silfreed.net/download/repo/

I'd appreciate it if someone could look over my current packages/spec files 
and let me know what my best step is to start contributing some of these back 
in.

Also, currently I just use VMWare to build these packages on multiple distros 
that I currently use at my work, and the vmware just runs on my laptop.  Of 
course, this means that they're only built for i386 as vmware won't run on 
x86_64 (for me, at least, reliably).  Are there any tools or build systems 
that might be made available for building these packages a little more 
automatically?
