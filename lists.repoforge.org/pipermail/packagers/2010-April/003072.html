I couldn't find any RPM distributions of Apachet Rivet, so I've
created my own spec file for packaging the current development
snapshots of it.  My spec file can be found on github:

http://github.com/bovine/rivet-rpm

Unfortunately, building from their development snapshots is currently
the best option since the last official release (0.5.0) was made in
Mar 2005.  The source code currently internally identifies itself as
0.8.0, so I'm using that with the snapshot timestamp as the release
suffix.

Any thoughts on including this in rpmforge?
