 From Disctionary:
ex·clude  (k-skld)	tr.v. ex·clud·ed, ex·clud·ing, ex·cludes

1. To prevent from entering; keep out; bar: a jar sealed to exclude 
outside air; an immigration policy that excludes undesirables.

2. To prevent from being included, considered, or accepted; reject: The 
court excluded the improperly obtained evidence.

That means that when you exclude some package you can not list it or 
install it.

Opposite from exclude from repository is to allow it to be installed, so 
you need to remove those packages from exclude list and then you can 
install them.
