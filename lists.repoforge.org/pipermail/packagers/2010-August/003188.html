work for something that is not used yet and has not been used in years.

wrong (often because upstream contact changed in the past).

Is anyone in charge of a package at rpmforge? If i would ask the person

less updates then I do now. My understanding of collaboration is that if
I see something wich i can fix myself i do it instead of sending my
changes to someone else...

Well that should be easily doable i guess. You probably have a script
extracting the source url from the spec and downloading the file
somwhere in you build process. Simply add a hash check after the
download.
