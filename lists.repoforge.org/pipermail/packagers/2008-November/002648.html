Hi,

Most packages contain those small files inline, see for example the 
dovecot.spec file. They use the 'EOF' approach.

Other packages use separate files which are checked in the svn repository in 
the same directory, for example:
http://svn.rpmforge.net/svn/trunk/rpms/collectl/
http://svn.rpmforge.net/svn/trunk/rpms/hylafax/
http://svn.rpmforge.net/svn/trunk/rpms/kannel/
For example they contain an <name>.logrotate file which 
becomes /etc/logrotate.d/<name> or an <name>.init file which becomes the 
file /etc/init.d/<name>.

kind regards,
Dries
