Dear RPMforge packagers,

I would like to suggest that the hdapsd package for EL6 be updated to the latest version (20090401), available from
http://sourceforge.net/projects/hdaps/
This userland daemon is required to use the hdd active protection system in ThinkPad notebooks.
The reason for this request is that the daemon currently available from RPMforge (a 2007 version) is unable to work with the hdd driver in the EL6 kernel (which, in contrast to EL5, supports disk head parking without patching!).
On my ThinkPad running RHEL6 (eval version), the package compiles without any issues (./configure; make; make install), and appears to work perfectly.
Would be great to see this update on RPMforge ;)

Thank you in advance.
Oliver
