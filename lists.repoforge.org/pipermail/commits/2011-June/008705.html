  phpmyadmin: fork phpmyadmin-php53 for EL5

Unfortunately, php53-* subpackages shipped with RHEL5 do not have the right
provides (php-mbstring etc.), so it is impossible to install phpmyadmin together
with php53. It seems that creating a separate phpmyadmin-php53 package is the
only way to go.
