I'm having the same exact issue.  If you don't want to wait for the
repository you can easily create the rpm yourself... apologies if you
already know this.  What I did was:

First setup your own rpmbuild environment (in your home directory)
sudo yum install fedora-rpmdevtools (installs required packages)

as non-root:
fedora-buildrpmtree (creates rpmbuild directories)

Download the latest NDISWRAPPER archive

as non-root:
rpmbuild -ta ndiswrapper_archive_you_downloaded (builds the ndiswrapper rpms)

Now you can install the two rpms which will be in the  
rpmbuild/RPMS/i386 or x86_64 directory (depending on what you're  
running).  As root use the rpm -i
command. To remove, use the rpm -e command.

Remember, only be in root to install the packages.  Not to build.

Hope this helps...
