On Wed, 20 Jun 2007, Schubert, Todd wrote:

> I have installed rpmforge-release-0.3.6-1.el5.rf so that I can receive
> updates of some non Red Hat packages that I have installed in my
> environment.  What I am finding though is that yum is updating Red Hat
> distributed packages with rpmforge ones.

...

> Is there any way to prevent this other than removing rpmforge from the
> repos?

You mean something like:

[Postgrey]
name=Postgrey Repository
baseurl=http://www.lfarkas.org/linux/packages/centos-$releasever/$basearch/
gpgcheck=0
enabled=0
#exclude=*
includepkgs=postgrey


See the yum.conf manual page to lear more about those repo file options.
