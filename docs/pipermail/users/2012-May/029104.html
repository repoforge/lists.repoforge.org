Hi,
I'd like to inform you that a new mail-list on mirroring has been
created. This list has been created to handle our communication on
Repoforge mirroring topic. We are in process of creating a new mirror
network for Repoforge project. We have prepared initial version of
Repoforge mirrors policy.

You can find some information on Repoforge mirrors at:
http://repoforge.org/faq/

If you are interested in becoming Repoforge mirror, please let us know by
email and join the mail-list at:
http://lists.repoforge.org/mailman/listinfo/mirrors

We currently have 12 active public mirrors and like to grow the number
significantly.
