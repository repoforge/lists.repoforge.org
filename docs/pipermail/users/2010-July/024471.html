Hello,

I'm the developer of GoAccess which is an open source real-time Apache web
log analyzer and interactive viewer that runs in a terminal and provides
fast HTTP statistics for system administrators that require a visual report
on the fly.

GoAccess can parse both of Apaches' log formats, the Common Log Format and
the Combined Log Format

License: GPLv2

Dependencies:
libgeoip1 (>= 1.4.7)
libglib2.0-0 (>= 2.16.0)
libncurses5 (>= 5.7)

More information about the project: http://goaccess.prosoftcorp.com/

Gerardo
