I've been doing some recent work with Musicbrainz.  It's an adventure. You
*cannot* use contemporary mod_perl, you must currently use mod_perl 1.30,
which is wildly out of date. I'm leaving those modules out of my request, to
avoid conflicts with OS packages for right now.

But these are necessary for that project, and I'd *love* to get them into
your repository. I've done all these with cpan2rpm, and would be happy to
send you the SPEC files and any patches.

   DBD::Pg, version 1.32, due to Musicbrainz compatibility issues with more
recent DBD::Pg versions. Perhaps this needs an alternate name, such as
perl-DBD-Pg_1.32-1.32-*.src.rpm?

  Apache::AuthDigest
  Apache::DB
  Apache::Sandwich
  Apache::Stage
  Apache::Cache::Memcached
  HTTP::Lite
  Language::Guess
  libapreq
  Musicbrainz::Client
  Musicbrainz::Queries
  RDFStore
  UUID

There are some additional non-perl modules used, as follows:

  PyLucene
  python-PygreSQL
  python-PyLucene
  python-unac
  unac
