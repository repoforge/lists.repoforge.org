Not at all, when you run:

spamassassin -D --lint 

you'll see the same "require failed" message, yet that doesn't mean it's
broken and it doesn't mean you have to install it.

Just install perl-Mail-SPF from rpmforge, make sure the loadplugin line in
your init.pre file is uncommented, and then run your -D on sa-update or
spamasssassin to see that it's loaded.

You don't need anything else after that.
