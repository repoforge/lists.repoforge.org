Please do not get confused by upstream version that is 0.98.1 (note the dot
before the 1) and package version 0.98-2 (not the dash before the 2). 

 

RPM version are created following the rules that are explained quite well in
http://docs.fedoraproject.org/en-US/Fedora_Draft_Documentation/0.1/html/RPM_
Guide/ch09s03.html

 

In short terms -2 means that the SPEC file was changes not the content of
the tarball of the upstream release.
