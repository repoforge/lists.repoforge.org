Hi:

python-abi was removed in favor of supporting Python on EL3.  Now, 
Requires: python = %{python_version}

This macro actually brings in the full version: 2.4.1 on FC4.  2.4.2 was 
just released upstream.  I am not sure when or if fedora will release 
this version.

However, a python package really only needs to depend on a specific 
minor release of python: 2.4, rather than on a micro release: 2.4.1 vs 
2.4.2.

Just FYI.
