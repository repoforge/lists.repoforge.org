Armin,

the issue seems to be that XML::LibXML::Reader only exists in XML- 
LibXML >= 1.62, and RHEL5 ships with XML-LibXML-1.58, so you can't get  
XML::LibXML::Reader in RHEL5 without overwriting a system package,  
which we try not to do.

however, looking at the code for Image::Info, it appears to be able to  
function with *either* XML::LibXML::Reader or XML::Simple, so we need  
to fix the dependencies in this package.  i've pushed the necessary  
changes.
