Hi

There's a small version of the Vi text editor called nvi.  It's
essentially the one that shipped with 4BSD.
http://www.bostic.com/vi/

It's already in Debian, and I think it's the same one the use
as /usr/bin/vi on FreeBSD.

Please find attached an RPM spec file and patch that makes it build.

The programs are called /usr/bin/nvi and /usr/bin/nview so they don't
clash with the versions Vim provides.
