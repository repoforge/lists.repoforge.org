Here's a third version that installs the three binaries (nvi, nview, and
nex) in /bin instead of /usr/bin.  This allows the user to run nvi even
if /usr is not available (e.g. if the system has crashed and /usr is not
mounted).

Note that the alternatives commands in the post install section only put
symlinks in /etc/alternatives.  They do not modify /bin/vi or the Vim
installation in any way.

The build patch is unchanged.
