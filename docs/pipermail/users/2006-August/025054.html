dag used a cool trick in the setup instructions at 
http://dag.wieers.com/home-made/apt/FAQ.php#B by feeding the 
rpmforge-release URL to the rpm command directly.  Small problem with 
that though: the provided URL for the package redirects to apt.sw.be, 
and the rpm command (at least the one in RHEL 4) doesn't know how to 
follow a redirect, so it errors out.

So of course I used wget to download it and install locally, but just 
thought I'd mention it since it makes the instructions look bad. :)

found out about the list) thanks to everyone contributing, it's a great 
resource!
