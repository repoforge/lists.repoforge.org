Hi guys,

I've done some work with the author of VoiceChatter and here are some
patches to apply to the spec for VoiceChatter.

Voicechatter-1.4.2_init-script-new.patch should replace the existing
init-script patch. It adds the new variables for some new server manager
options.

Voicechatter-1.4.2_VCServerManager.patch fixes a few issues - mainly the
requirement for the server executable to be in the same directory as the
config files.

These will probably both be outdated when voicechatter 1.4.3 is released.
