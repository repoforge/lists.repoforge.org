Hi there,

I have finally found some spare time to update my smplayer patchset. Now
that (thanks to Dag!) we have newer mplayer it really makes sense.
Limitations are as usual: no Socks5 proxy, no HTTPS support.

Most important changes since 0.6.6:

* Added support for TV 

* New option to enable a global volume

* Added experimental (and uncomplete) support for dvd menus

* Now loading an external subtitle file doesn't require to restart the
mplayer process (except for idx/sub subtitles)

* Added two options (in the audio and subtitle menus) to allow the user
to enter the audio and subtitle delay (in milliseconds).
