Hello,

As topic says:
file /usr/share/man/man3/Test::Builder::Tester.3pm.gz from install of
perl-Test-Builder-Tester-1.01-1.noarch conflicts with file from
package perl-5.8.8-27.el5.x86_64

Fix:

$ diff -u perl-Test-Builder-Tester.spec perl-Test-Builder-Tester.spec.orig
--- perl-Test-Builder-Tester.spec	2011-01-28 16:09:42.271826065 +0000
+++ perl-Test-Builder-Tester.spec.orig	2011-01-28 16:10:00.680172906 +0000
@@ -48,8 +48,8 @@
 %files
 %defattr(-, root, root, 0755)
 %doc CHANGES MANIFEST META.yml
-%exclude %{_mandir}/man3/Test::Builder::Tester.3pm*
-%exclude %{_mandir}/man3/Test::Builder::Tester::*.3pm*
+%doc %{_mandir}/man3/Test::Builder::Tester.3pm*
+%doc %{_mandir}/man3/Test::Builder::Tester::*.3pm*
 %dir %{perl_vendorlib}/Test/
 %dir %{perl_vendorlib}/Test/Builder/
 %dir %{perl_vendorlib}/Test/Builder/Tester/
