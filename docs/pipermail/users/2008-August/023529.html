Hello,

I have built rpm package for the newly famous light download
accelerator for Linux called Axel.

Below is my axel.spec file and will attach the el5.x86_64 rpm file I
already built and will also list the rpmbuild command parameters I
used for building.


# # axel.spec # #
%define name    axel
%define version 1.1

Name:           %{name}
Version:        %{version}
Release:        1%{?dist}
Summary:        A light download accelerator for Linux
Packager:       Ahmed Medhat <ultimatetux at gmail.com>
Vendor:         Dag Apt Repository, http://dag.wieers.com/apt/

Group:          Applications/Internet
License:        GPLv2
URL:            http://axel.alioth.debian.org/
Source:         %{name}-%{version}.tar.gz
BuildRoot:      %{_tmppath}/%{name}-%{version}-%{release}-root-%(%{__id_u} -n)
BuildRequires:  gcc, binutils


%description
Axel tries to accelerate the downloading process by using multiple
connections for one file. Starting from version 0.97, the program can use
different URL's for one download as well. The program tries to be as light
as possible (25-30k in binary form), so it might be useful as a wget clone
on byte-critical systems.

%prep
%setup -q

%build
./configure --prefix=%{_prefix} --etcdir=%{_sysconfdir}
--mandir=%{_mandir} --i18n=0

make %{?_smp_mflags}

%install
rm -rf $RPM_BUILD_ROOT
make install DESTDIR=$RPM_BUILD_ROOT

%clean
rm -rf $RPM_BUILD_ROOT

%files
%defattr(-,root,root,-)
%doc
/usr/bin/axel
/etc/axelrc
/usr/share/man/man1/axel.1.gz

%changelog
* Mon Jul 28 2008 Ahmed Medhat <ultimatetux at gmail.com>
- Initial RPM creation
# # axel.spec # #


The exported RPM file is axel-1.1-1.el5.rf.x86_64.rpm

Command used to build: rpmbuild -bb --clean --without debuginfo
--target=x86_64 --define "dist .el5.rf" --define "_prefix /usr"
--define "_sysconfdir /etc" --define "_mandir /usr/share/man"
axel.spec

Note: I would like to maintain/followup with the package in the future.



Thanks, /Ahmed Medhat/
