I am willing to (would like to) be the maintainer for this package.

HylaFAX(tm) is a enterprise-strength fax server supporting
Class 1 and 2 fax modems on UNIX systems. It provides spooling
services and numerous supporting fax management tools.
The fax clients may reside on machines different from the server
and client implementations exist for a number of platforms including
windows.

SPEC: http://osdn.dl.sourceforge.net/sourceforge/hylafax/hylafax.spec
SRPM:
http://osdn.dl.sourceforge.net/sourceforge/hylafax/hylafax-5.2.0-1.src.rpm
<http://osdn.dl.sourceforge.net/sourceforge/hylafax/hylafax-5.1.11-2.src.rpm>
Website:  http://hylafax.sourceforge.net

(The spec is for the older, 5.1.11-2 release, but nothing differs other
than the version info.)
