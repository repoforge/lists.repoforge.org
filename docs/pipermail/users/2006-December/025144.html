There seems to be a circular obsolescence between the 'perl-rrdtool'
from Dries/RPMForge and 'rrdtool-perl' from Fedora Extras for FC6.
Install one and the other obsoletes it.  Interestingly enough, it only
seems to actually work with 'pup'--the yum CLI throws a Python
exception: 'RuntimeError: maximum recursion depth exceeded', after going
through a bunch of attempts to process the obsoletes parameters.  I
suspect the python or tcl bindings will do the same.
