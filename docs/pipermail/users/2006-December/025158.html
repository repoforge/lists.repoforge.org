> On Dec 6, 2006, at 10:16 AM, Dag Wieers wrote:
> 
> >> Can FC please remove the package version numbers from the  
> >> directories in
> >> /usr/share/doc and have a policy that each package should have a  
> >> directory in
> >> /usr/share/doc?  Using this directory has been used so seldom that  
> >> I rarely
> >> even look there anymore on RH based systems now.
> >
> > That would be a nice suggestion to make as well, especially because  
> > the
> > version number makes it impossible to have persistent links from eg.
> > documentation and websites.
> 
> alternately, in %post do a 'ln -s %{_defaultdocdir}/%{name}-%

in doc/ since the beginning and I cannot imagine it will change.  I bet
there's even a Bugzilla item for it...
