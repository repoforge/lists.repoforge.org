Attached is a modified .spec file to build Subversion 1.6.3 on EL4.

The resulting RPMs do not have any external dependencies on specific 
versions of neon, apr, apr-util, sqlite, etc.  No other RPMs are 
required for the build environment.  Although python 2.4 is included in 
the source RPM and built at compilation time (for "el4" builds only), it 
is not installed as a separate RPM so as to keep the build environment 
clean.

Binary RPMs are available at:
http://lxpro.com/subversion/1.6.3/el4/RPMS/

Original (1.6.3-0.1) and modified SPEC files are available at:
http://lxpro.com/subversion/1.6.3/el4/SPECS/

Greg Bailey
