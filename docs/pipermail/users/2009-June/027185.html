It looks like gdal's dependencies require libgeos.so.2 which ships with
an older version of geos, while the geos you're trying to install is the
newest

You could be able to force the older geos installation, then install
gdal, like this:

yum install libgeos.so.2
yum install gdal

I'm no expert on the matter, just an assumption
