Hi!

It's been broken for a few years I guess. I remember I posted about
that, and then somebody else and you're like the third or whatever.

Concerning Audacious, it's virtually impossible to cleanly update it to
the latest version, because it needs newer GTK and the other libs. I
remember that I once managed to compile it from source after patching
the m4 scripts to lower the requirements for GTK, downloading the latest
versions of the libs it needs and statically linking Audacious against
them, but I never managed to make a package out of it, because it was so
much pain.

As far as xmms + flac are concerned, this is a known issue and quite
easy to fix, but nobody cares. I the end, I managed to build flac + flac
plugin, as well as a special patched version of xmms with recode support
myself. You can find the binaries and SRPMs on my web page (i386 only):

http://rpm.zaytsev.net/i386/

Hope that helps,
