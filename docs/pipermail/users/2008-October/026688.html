> Hi,
>
> I hat to rpm -e vlc to be able to successfully perform yum upgrade in the
> last weeks. My system is CentOS 4.7. Of course I want vlc back, but get

> Error: Missing Dependency: libdirect-1.0.so.0 is needed by package vlc

>

If you want to rebuild the src rpm:
You need to define 'dtag' so it uses the correct buildrequirements for 
centos4:

rpmbuild --define 'dtag el4' --rebuild yoursrcrpm
