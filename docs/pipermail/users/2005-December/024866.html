You have the atrpms ffmpeg installed which does not have the provides
libavutil declared in it like RPM forge's does.  So what I did to get
around this is I disabled Atrpms in my yum repos.
then ran:
rpm -e ffmpeg --nodeps
yum install ffmpeg
yum install kino

then you can re-enable Atrpms.  But you can not update until he adds the
provides to his rpm.
