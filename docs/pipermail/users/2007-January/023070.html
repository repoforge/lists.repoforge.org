I use MailScanner (www.mailscanner.info) on several mail servers. 
MailScanner relies on this particular perl-DBI version (v1.50).

When installing MailScanner, I use the install.sh script provided in the 
package which compiles the src RPM's supplied with the MailScanner tarball 
(I've supplied the list below).

Many of the compilations fail to create the binary RPM's so they don't get 
installed, so I source as much as I can from rpmforge and Dries.

Installing the pre-requisite RPM packages from you guys allows the 
install.sh script to say "package is installed, skipping" and ending in a 
successful MailScanner installation.
