The missing python-cddb for exaile came up recently for CentOS 6. I note
that the python-cddb package does not compile properly for Scientific Linux
6, due to changes in Python package generation under Python 2.6. I've
updated it to support a "_without_info" option for older releases, and to
include the newly genearated
/usr/lib64/python2.6/site-packages/CDDB-1.4-py2.6.egg-info file. I've also
tweaked it so it stops kvetching about double-included files. (The *.pyo
files were included as normal and as '%ghost' files).

It's available at https://github.com/nkadel/puthon-cddb-1.4-srpm, and
should be easily copied onto the repoforge spec file.
