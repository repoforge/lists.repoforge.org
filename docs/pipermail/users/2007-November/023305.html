Hi,

Here are RPMs of innotek's VirtualBox 1.5.2 OSE (Open Source Edition) 
<http://www.virtualbox.org> product, compiled for CentOS 4.X and CentOS 
5.X (i386).

These packages rely on the the DKMS 
<http://linux.dell.com/dkms/dkms.html> system to automatically build and 
install the kernel driver required by VirtualBox for the currently 
installed kernel.

Known issues with these RPMs:

    * The dkms, xerces-c, and xalan-c RPMs are required for runtime
      operation.
    * The xerces-c, xerces-c-devel, xalan-c, and iasl RPMs are required
      to build VirtualBox-OSE.
    * SMP kernels don't seem to boot properly and hang the guest.
    * I'm assuming that the User Manual
      <http://www.virtualbox.org/download/UserManual.pdf> and the
      VBoxGuestAdditions_1.5.2_OSE.iso
      <http://www.virtualbox.org/download/1.5.2/VBoxGuestAdditions_1.5.2_OSE.iso>
      files can be freely redistributed. If not, they can easily be
      removed from the RPMs.

Please contact me <mailto:gbailey at lxpro.com> if you have any questions 
or feedback. Thanks!

-Greg
