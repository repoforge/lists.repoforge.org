Hi,

Could you please update nagios-nrpe to the current stable release v2.12?
It adds support for nrpe -u command line switch and multiline plugin
output (among a lot of other new features) we'd like to use.

It seems the current (SVN r6709) nagios-nrpe.spec for nrpe-2.5.2 works fine
with nrpe-2.12, too, at least on our RHEL 5.3 x86_64 servers.

A trivial patch against SVN release 6709:

--- nagios-nrpe.spec.svn-r6709  2006-12-12 01:44:32.000000000 +0200
+++ nagios-nrpe.spec    2009-03-03 09:39:55.000000000 +0200
