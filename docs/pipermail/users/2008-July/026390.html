$ rpm -qf /usr/lib64/perl5/5.8.8/x86_64-linux-thread-multi/Time/HiRes.pm
perl-5.8.8-10.el5_2.3.x86_64

$ rpm -qf /usr/lib64/perl5/5.8.8/x86_64-linux-thread-multi/Digest/MD5.pm
perl-5.8.8-10.el5_2.3.x86_64

it seems both of these perl modules are included in the standard perl 
package on centos 5.2, so you don't need to install them from rpmforge.

As to yum wanting to install perl.i386, hasn't it been described as a 
yum bug on the centos list?
