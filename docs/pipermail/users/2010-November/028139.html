  perl-DBI is missing from the repository. (As i currently discovered 
perl-DBI is in the extras repository which does't get enabled by the 
rpmforge-release.rpm on the website).

[root at centos ~]# perl -e 'require DBD::SQLite'

/usr/lib/perl5/vendor_perl/5.8.8/i386-linux-thread-multi/DBD/SQLite.pm 
line 5.

[rpmforge]
name = Red Hat Enterprise $releasever - RPMforge.net - dag
#baseurl = http://apt.sw.be/redhat/el5/en/$basearch/dag
