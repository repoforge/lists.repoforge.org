I've tested Mercurial 1.7.1 with the existing SPEC file, and all that is required is to change the version.


diff --git a/mercurial.spec b/mercurial.spec
--- a/mercurial.spec
+++ b/mercurial.spec
@@ -10,7 +10,7 @@

 Summary: A fast, lightweight Source Control Management system
 Name: mercurial
-Version: 1.6.4
+Version: 1.7.1
 Release: 1%{?dist}
 License: GPLv2+
 Group: Development/Tools


The update will greatly improve sub-repository functionality, among other things:
http://mercurial.selenic.com/wiki/WhatsNew#A1.7.1_.282010-11-15.29
