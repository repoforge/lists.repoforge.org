> When my system was booting (a rare event), I saw a message about

> And Madwifi seem to be working just dandy.

Try using CTRL-S (to suspend) and CTRL-Q (to unsuspend) while it's booting.

Failing that, you can modify the startstop script of madwifi and use "tee" to
output to console and a file.
