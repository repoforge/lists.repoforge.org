Munin (http://munin.projects.linpro.no/) is a monitoring toolkit that utilises rrdtool and is complementary to nagios.

The munin docs actually reference Dags RPM repository for the dependencies (rrdtool, various perl modules), and nagios is also on rpmforge, so it'd be great if munin itself was made available in the same place.

There are two packages: munin (server) and munin-node (client).
