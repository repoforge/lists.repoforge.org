On Fri, 27 Apr 2007, Tom G. Christensen wrote:

> Any particular reason this is not built?

Yes, the sendmail that comes with EL2 does not have milter support (and 
therefor clamav-milter cannot be build against it).

If you however replaced the official sendmail with a newer one, you can 
rebuild that package yoursellf by adding rpmbuild --with-milter.

 
> I don't use it myself but it has built without error since 0.75.1 when
> configure was taught how to deal with sendmail 8.11 milter (and now el2 is
> officially at 8.12 so that's not even an issue anymore.)
