Hallo packagers

i have update my nagiosgrapher packages for the new RC5. It should be
now possible to get it up and running with only the following steps:

- install RPM
- add /etc/nagios/ngraph-commands.cfg in nagios.cfg
- start nagios_grapher with init-script

This is of course if you have rpmforge nagios running.

You can find spec-file and SRPM at:

http://members.financial.com/~cmr/rpmforge/nagiosgrapher/
