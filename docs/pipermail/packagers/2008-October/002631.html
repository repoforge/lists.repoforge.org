Up to now I have packaged up my own cacti, plugin arch and plugins:
# rpm -qa | grep ^cacti
cacti-plugin-1.4-1
cacti-plugin-hostinfo-0.2-1
cacti-0.8.7a-2
cacti-plugin-ntop-0.1-1
cacti-plugin-update-0.4-1
cacti-docs-0.8.7a-2
cacti-plugin-thold-0.3.9-1
cacti-plugin-settings-0.3-1

Yum works fine - from my own repo ;-)

cacti-plugin is a hack though: It applies the orginal plugin arch patch
on top of the cacti tree (uuurgghh).
That works most of the time but uninstall has problems sometimes
removing the patch.
Having the install scriptlets edit cacti/include/config.php to integrate
the plugins is also not optimal ...

Been waiting for cacti 0.88 to include the plugin arch:
http://www.cacti.net/roadmap.php

The cacti-plus effort might not be worth it with just a few months to go
until cacti comes with the plugin arch integrated.
