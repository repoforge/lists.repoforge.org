Because I may be using this for a customer as well, and this pretty much 
proves I wrote it before I use it at the customer site :)

Also, I don't think this is quality code. More for tinkering with and 
getting some experience with initramfs.

We have a big issue where it works for simple initramfs archives (the 
minimal one), but it consistently fails for big ones (40MB). While it does 
work for the RIPLinux initramfs which is 45MB.

So I prefer to find the cause, play with it some more and then think about 
how we want to use this. Obviously the problem with what we are doing here 
is not trying to reduce the size, but merely making it based on CentOS.

Also, I would like to use repository metadata and extract files directly 
from RPMs, instead of relying on the host OS. On the other hand, the 
current script should work on any OS, which has its advantages as well.
