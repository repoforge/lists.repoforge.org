If you would like to fix up your public key settings, you first need to
generate a dedicated private/public key pair for github or use an
existing one.

Then you need to add this public key to github's interface, so that it
knows that this key is associated with your account.

There is a step-by-step guide over here:

    https://help.github.com/articles/generating-ssh-keys

Skip the part that has to do with generating a new key pair if you would
like to re-use an existing key pair. Also, of course, feel free you use
any other ways of copying the public key to the clipboard ;-)
