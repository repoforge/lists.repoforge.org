Hello,

A new version of Collectd (5.2) is available for a while
http://collectd.org/.
Is there a possibility to get new collectd-*.rpm packages built and added
to RPMforge repository soon, by any chance ?

Also, it would be great to have a collectd varnish plugin enabled in it.

To build collectd varnish plugin, you just need to have varnish-libs-devel
and varnish-libs packages installed (on RH/Centos/Fedora). Collectd build
configuration script will detect it and build varnish plugin automatically
(though, you may need then to update collectd.spec file - add
'%{_libdir}/collectd/varnish.so' to %files section).
