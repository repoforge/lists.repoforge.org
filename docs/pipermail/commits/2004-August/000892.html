++dnl AS_LINUX()
++dnl
++dnl this macro adds the options
++dnl --with-linuxdir        to specify a kernel build tree location

++dnl LINUX_DIR
++dnl   The directory where the Linux source resides.
++dnl CONFIG_FILE
++dnl   The Linux config file
++dnl LINUX_ARCH
++dnl   $(ARCH) in kernel Makefiles
